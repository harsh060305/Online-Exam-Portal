# Online-Exam-Portal

A web-based **Online Examination System** built using **Spring Boot**, **Spring MVC**, **Spring Security**, **Hibernate**, **MySQL**, **JSP**, **Bootstrap**, and **Maven**. The system allows administrators to manage exams, questions, students, and results while providing students with a secure platform to take online examinations.

---

## 🚀 Features

### 👨‍💼 Admin Module
- Secure Admin Login
- Dashboard
- Manage Students
- Manage Exams
- Add, Update, Delete Questions
- Assign Questions to Exams
- View Student Results
- Manage User Profiles

### 👨‍🎓 Student Module
- Secure Login & Registration
- View Available Exams
- Start Online Exam
- Submit Exam
- View Exam Results
- Update Profile
- Upload Profile Picture

## Run Project

Using Maven

```bash
mvn spring-boot:run
```

or

```bash
mvn clean install
```

Run from IntelliJ IDEA or Eclipse as a Spring Boot Application.
